💻 RADEG PANELS

A Hacker-Style Web Interface built using HTML, CSS, and JavaScript

---

🚀 Overview

RADEG PANELS is a futuristic, hacker-themed web project designed to simulate a mini cyber control system. It provides an interactive interface with login/register system, command-based chat, and system-like tools such as CPU/RAM stats.

This project is made for learning, creativity, and fun — giving users a feel of a real hacking dashboard 😎🔥

---

⚡ Features

🔐 Authentication System

- Login and Register interface
- Clean UI with centered inputs
- Fake authentication logic for simulation

⏳ Loading Screen

- 10-second animated loading
- Dynamic text like:
  - Starting...
  - Accessing GitHub...
  - Encrypting...
  - Decrypting...
  - Welcome!

📊 Dashboard (Main Panel)

- Sidebar navigation menu
- Toggle menu system
- Logout functionality

🧠 CPU & RAM Monitor

- Displays fake CPU and RAM usage
- Updates every second
- Center-aligned large display

💬 Command Chat System

Type commands in the chat box and get responses from Server AI

Supported Commands:

- "help" → Shows all commands
- "scan" → Simulates system scan
- "hack" → Displays fake hacking process
- "about" → Info about the panel + support
- "stats" → Info about CPU/RAM usage
- "map" → Shows "Under Development"
- "clear" → Clears chat
- "logout" → Returns to loading screen
